FAQ Sammlung für das IPTV List Updater Plugin!!!
Stand: 15.06.2013

1. Für welche Systeme ist das Plugin gedacht?!
    Das Plugin sollte auf allen OE2.0 Images funktionieren. OE1.6 Images 
    unterstützen keine IPTV Bouquets !
	
2. Was kann ich machen, wenn ich das Plugin nicht installieren kann?
    Entpacke das Plugin z.B. mit 7zip. Kopiere alle Files mit einem FTP 
    Programm (z.B. Filezilla) auf deine Box und setzte die Rechte der 
    Scripte im Ordner 
    "/usr/lib/enigma2/python/Plugins/Extensions/IPTV-List-Updater/scripts" 
    auf 755.

3. Welche Boxen können MMS Streams abspielen?
    Stand: 15.06.2013:
    VU+ haben kein mms support! Nur alle Xtrend, dm500hd,800se,7020hd 
	
4. Was muss ich tun, um zusätzliche Streams in die Listen zu bekommen?
    Sucht entsprechende Linklisten im Internet. Testet sie und postet sie in 
    einem der zahlreichen Threads. 
    Sicherer ist es, wenn ihr satinfo oder HasBahCa oder mich im VU+ Forum 
    über PM kontaktiert.
    ... nur so ist die Wahrscheinlichkeit am höchsten, dass es klappt.
	
5. Wie kann ich Bouquets wieder löschen?
    Einzelne Bouquets könnt Ihr nach Beendigung des Plugins wieder löschen:
    Pfeil runter => Favoriten (blaue Taste) => auf das zu löschende Bouquet
    gehen => Menü => Eintrag entfernen
    Beim nächsten Aufruf des Plugins werden die Bouquets wieder angelegt, 
    diese wieder gewählt wird zum installieren!
